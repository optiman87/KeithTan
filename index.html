Data Cleaning with SQL

Given the recent global layoffs, I found it worthwhile to analyze the prevailing trends. Utilizing the layoff dataset from layoff.fyi, I imported the data into MySQL. I then performed data cleaning using SQL to remove duplicate data, standardize field values and remove unnecessary or null rows and columns. The clean data was exported to a csv file. The file was then imported into Tableau to create the World Layoffs Dashboard .

I aim to monitor the prices of various cryptocurrencies in CAD, along with their percentage changes over time.
