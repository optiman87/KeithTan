How to disable NAT Gateway for Azure Databricks

When creating an Azure Databricks workspace, Microsoft Azure also creates a NAT Gateway, a VNet (Virtual network) and other resources to support the workspace. Most of these resources cost almost nothing, except for NAT Gateway, which costs 0.062 CAD (0.045 USD) per hour just to exist. This means that as long as the workspace exists, users will be charged 44.64 CAD per month even if Azure databricks has zero clusters turned on and no workflows running....
